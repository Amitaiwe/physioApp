# יומן טיפולים — PWA אישי

## מה זה
אפליקציית PWA להתקנה על מסך הבית באייפון, לניהול יומן טיפולים, תיעוד קולי + תמלול, ומעקב סטטוס.
כל הנתונים (טיפולים, מטופלים, הקלטות, תמלולים) נשמרים **מקומית בלבד** במכשיר (IndexedDB), בלי שרת ובלי ענן.

## איך זה בנוי
- קובץ סטטי לחלוטין — בלי שלב build. React, htm, Dexie ו-JSZip נטענים ב-`<script type="module">` ישירות מ-CDN (esm.sh).
- **htm** מאפשר syntax דמוי-JSX (`html\`<div>...\`</div>``) בלי טרנספיילר.
- Service Worker (`sw.js`) שומר את קבצי האפליקציה עצמם לעבודה אופליין. הנתונים נשמרים בנפרד ב-IndexedDB ולא נמחקים כשה-cache מתעדכן.
- תמלול: Web Speech API (מובנה בדפדפן/במכשיר), ללא שליחת אודיו לענן — כפי שביקשת.

## פריסה (GitHub Pages, כמו הפרויקטים הקודמים שלך)
1. צרי repo חדש (למשל `physio-diary`) תחת amitaiwe, והעלי את כל תוכן התיקייה הזו לענף `main`.
2. הפעילי GitHub Pages: Settings → Pages → Deploy from branch → `main` / root.
3. פתחי את הכתובת שתתקבל (`amitaiwe.github.io/physio-diary`) **בספארי באייפון**.
4. שיתוף ⬆️ → "הוסף למסך הבית" — האפליקציה תותקן ותיפתח כמו אפליקציה רגילה, בלי סרגל כתובות.

**חשוב:** `getUserMedia` (הקלטה) ו-Service Worker דורשים HTTPS — GitHub Pages נותן את זה אוטומטית.

## מה נבדק/צריך בדיקה אצלכם באייפון
- איכות הקלטה + Web Speech API בעברית — אישרת שנתחיל מזה, כדאי לבדוק כמה משפטים אמיתיים ולראות אם האיכות מספיקה. אם לא — נעבור לתמלול ענן עם מסך הסכמה מפורש.
- תמיכת OPFS/גודל אחסון — כרגע ההקלטות נשמרות כ-Blob בתוך IndexedDB (פשוט ואמין ל-MVP). אם ייצטבר נפח גדול מאוד של הקלטות, אפשר לשדרג ל-OPFS בהמשך.
- הרשאות מיקרופון — ספארי תשאל הרשאה בהקלטה הראשונה.

## מבנה קבצים
```
index.html            מעטפת האפליקציה
manifest.webmanifest   הגדרות PWA (שם, אייקונים, מצב standalone)
sw.js                  Service Worker לעבודה אופליין
css/app.css            עיצוב (טוקנים, טיפוגרפיה, חתימת עיצוב)
js/db.js               סכימת IndexedDB (Dexie)
js/app.js              נקודת כניסה + ניתוב
js/utils/               פונקציות עזר (תאריכים, הקלטה, מחיקה אוטומטית)
js/views/                המסכים: Home, Calendar, TreatmentForm, TreatmentDetail, Stats, Backup
```
